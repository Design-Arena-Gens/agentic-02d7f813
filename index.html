Quantum CLI Encryptor

This project ships a secure Arch Linux command-line program that encrypts user messages with a quantum-sourced key derivation workflow. The CLI consumes entropy from the ANU QRNG API, derives keys via PBKDF2-HMAC-SHA512, and stores a deterministic 512-character ciphertext in <username>.txt.

Build

make

Run

./quantum_cli

Test

make test

The binary lives entirely in C with memory cleansing, strict password validation, and safe file replacement. Source code resides in src/ and tests in tests/.

Deploying via Vercel serves this reference page while the CLI continues to operate locally.
